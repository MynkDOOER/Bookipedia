
# Bookipedia <img width="28" height="28" alt="book" src="https://github.com/user-attachments/assets/9ed8e05b-3957-49e0-a077-0408174b16b0" />

Welcome to **Bookipedia**, a modern, sleek web application built with React.jsx that allows users to discover new books, view detailed information, and curate a personalized list of favorites.

## Features

- Search books by name or keyword
- View complete information about a book
- Add books to favorites
- Remove books from favorites
- Responsive design
- Fast search using debouncing.

## 🛠️ Tech Stack

* **Frontend:** React.js (JSX)
* **Routing:** React Router (for seamless navigation between Home, All Books, and Favorites)
* **Styling:** CSS / Custom Dark Theme UI

## 💻 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

* npm
```sh
npm install npm@latest -g

```



### Installation

1. Clone the repo
```sh
git clone https://github.com/MynkDOOER/Bookipedia.git

```


2. Navigate to the project directory
```sh
cd Bookipedia

```


3. Install NPM packages
```sh
npm install

```


4. Start the development server
```sh
npm start

```



The application will launch in your browser, typically at `http://localhost:3000`.
